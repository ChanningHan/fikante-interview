## Vue与React的区别
React
- 即简单也复杂
- JSX灵活
- 更像是工具库，UI编码提效
- 心智负担较高
- 生态更蓬勃，但也逼死选择困难症


Vue
- 更容易上手，更为用户考虑，注重易用性
- 模版编译分析优化
- 更像框架，在特定的模版、API下进行开发，更高的封装度进行提效。
- 心智负担较低

都是JS上层框架，本质还是JS，框架只是工具，应该更关注底层的JS。
基本没有什么是两者互相不能实现的，只是风格不一样罢了。
